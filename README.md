## Использованный стэк технологий

- **REACT VITE** - Vite был использован вместо Next.js так как разрабатываемая платформа не нуждается в SEO, то есть она предназначена для внутреннего использования без широкого охвата;
- **REACT QUERY** - Очень мощная библиотека для работы с кешом и запросами на бэкенд;
- **REACT ROUTER DOM** - Для маршрутизации;
- **GSAP** - Для анимаций (Пока не реализовано);
- **TIPTAP** - Для редактора разметки (Пока не реализовано);
- **TAILWIND** - Для стилизаций;
- **SHAD CN компоненты** - Для быстрой разработки МЖП проекта;
